 Silicon Engine

**2D game engine written in C++17 with OpenGL.**  
Fast, minimal, and scriptable with ArsenScript (your own programming language).

[![C++17](https://img.shields.io/badge/C%2B%2B-17-blue.svg)](https://isocpp.org/)
[![OpenGL](https://img.shields.io/badge/OpenGL-3.3-green.svg)](https://www.opengl.org/)
[![License](https://img.shields.io/badge/license-MIT-red.svg)](LICENSE)

---

## ✨ Features

**Cross-platform** (Windows / Linux / Android planned)
 **Scene system** (easy game states)
 **2D graphics** with OpenGL (fast rendering)
